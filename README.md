### Steps to install locally
- Fork the Repo
- Clone the Repo
  ```bash
  git clone https://github.com/hitesh181/quizzical.git
- Navigate to the directory
  ```bash
  cd quizzical

- Install necessary dependency and packages
  ```bash
   npm i
- Launch on local server with
  ```bash
  npm start
